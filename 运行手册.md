# PS 正反图 — 快速运行手册 v1.0

## 触发词
用户说 **"ps"** 或 **"PS批处理"**

## 运行命令
```bash
cd E:\Claude code\ps
python ps_batch.py
```

## 工作原理
1. 扫描 `D:\Semems\1AI` 下所有 DX 文件夹
2. 跳过已有 `白BW.jpg`/`黑BW.jpg` 的
3. 一次性打开正背两张图 → 运行动作 → 保存 → 关文档
4. 白T处理完自动处理黑T

## 依赖
```bash
pip install win32com
```

## 停止
用户说 **"x"** 或 **"停止"**，立即杀进程。
