# WB 贴图 PS 版 — 运行手册

## 安装依赖
```bash
pip install pywin32 pillow numpy
```

## 第一阶段：贴图生成

```bash
# 批量处理所有DX
python wb_sticker_ps.py

# 从指定DX开始（如DX0124）
python wb_sticker_ps.py DX0124
```

自动计时：每张图耗时、每款耗时、总耗时。

## 第二阶段：BW合成

```bash
python ps_batch.py [起始DX]
```

需要 PS 有 `正反图` 动作组。

## 预览黑BW图

双击 `D:\Semems WB\02_PROJECTS\01_CHECK\同步黑T.bat`，功能：

| 操作 | 效果 |
|------|------|
| 选择日期 | 切换不同日期的黑BW预览 |
| 点击图片 | 资源管理器打开对应03_UPLOAD文件夹 |
| 🗑 同步删除 | 删缩略图 → 03_UPLOAD原图送回收站 |
| ♻ 恢复 | 从回收站还原已删文件 |
| 🔄 刷新 | 重新生成缩略图（文件仍在原日期） |
| × 按钮 | 单张缩略图送回收站 |

> **注意**: 刷新不受文件修改时间影响，图片永远留在它被处理的那一天。

## 配置调整

编辑 `config.py`：
- `BACK_NEW`: 背图参数（缩放、旋转、位置）
- `FRONT_NEW`: 正图参数
