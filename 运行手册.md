# WB 贴图 PS 版 — 运行手册

## 前置准备
1. 安装 Python 3.x
2. 安装依赖:
   ```bash
   pip install pywin32 pillow
   ```
3. 确保 Photoshop 已安装并能正常启动
4. 检查 `config.py` 中的路径是否正确:
   - `BASE_TORSO`: T 恤模板所在文件夹（如 `D:\Semems\1胚衣`）
   - `SOURCE_BASE`: DX 项目根目录（如 `D:\Semems WB\02_PROJECTS`）


## 使用方法

### 批量处理所有 DX 文件夹
直接运行主脚本:
```bash
python wb_sticker_ps.py
```

### 处理单个 DX 文件夹
可以修改主脚本或使用临时测试脚本（如 `test_dx0051.py`）。


## 注意事项
1. **运行前请关闭 Photoshop 中已打开的文档**，避免 COM 报错 "application busy"。
2. 生成的图片会保存在 `DX*/03_UPLOAD/` 文件夹下，文件命名格式为 `DXxxxx_XX_cut_<白T/黑T>.jpg`。
3. 如需调整位置/大小/旋转角度，修改 `config.py` 中的 `BACK` 和 `FRONT` 配置即可。


## 验证结果
- 检查 DX0050、DX0051、DX0086 的输出图，确认贴图位置和效果符合预期。
