# WB 贴图 PS 版 — 运行手册

## 前置准备
1. 安装 Python 3.x
2. 安装依赖:
   ```bash
   pip install pywin32 pillow
   ```
3. 确保 Photoshop 已安装并能正常启动
4. 确保 Photoshop 中有 `正反图` 动作组，包含 `白` 和 `黑` 两个动作
5. 检查 `config.py` 和 `ps_batch.py` 中的路径是否正确:
   - `config.py`: `BASE_TORSO` 和 `SOURCE_BASE`
   - `ps_batch.py`: `BASE` 和 `PS_EXE`


## 完整使用流程

### 第一阶段：贴图（所有类型）
运行主脚本，处理 B、W 和 BW 类型的设计图：
```bash
python wb_sticker_ps.py
```

这一步会生成：
- **B类型**: `DXxxxx_B_白T.jpg`，`DXxxxx_B_黑T.jpg`（背面贴图）
- **W类型**: `DXxxxx_W_白T.jpg`，`DXxxxx_W_黑T.jpg`（正面贴图）
- **BW类型**: 同时生成上述4个文件（为第二阶段合成做准备）


### 第二阶段：BW 合成（仅 BW 类型需要）
如果有 BW 类型的设计图，运行原来的合成脚本：
```bash
python ps_batch.py
```

这一步会：
- 读取 `DXxxxx_W_*T.jpg` 和 `DXxxxx_B_*T.jpg`
- 调用 Photoshop 的 `正反图` 动作组
- 生成最终的 `DXxxxx_白BW.jpg` 和 `DXxxxx_黑BW.jpg`


## 注意事项
1. **运行前请关闭 Photoshop 中已打开的文档**，避免 COM 报错。
2. 生成的图片会保存在 `DX*/03_UPLOAD/` 文件夹下。
3. 如需调整位置/大小/旋转角度，修改 `config.py` 中的 `BACK` 和 `FRONT` 配置即可。


## 验证结果
- 检查 DX0050、DX0051、DX0086 的输出图，确认贴图位置和效果符合预期。
- 特别是 BW 类型，确认两个阶段都正常运行并生成最终的合成图。
