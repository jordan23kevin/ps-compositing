# WB 贴图 PS 版 — 运行手册

## 安装
```bash
pip install pywin32 pillow numpy
```

## 贴图生成
```bash
# 批量
python wb_sticker_ps.py

# 从指定DX开始
python wb_sticker_ps.py DX0124
```
自动跳过黑版文件（含`_黑`的_cut.png）。

## BW合成
```bash
python ps_batch.py [起始DX]
```
需要 PS 有 `正反图` 动作组。

## 黑T处理（反相后）
```bash
python process_black.py DX0151
```
独立脚本，由 ⮂ 按钮自动调用。也可单独运行。

## 预览黑BW图
双击 `D:\Semems WB\02_PROJECTS\01_CHECK\同步黑T.bat`

| 操作 | 效果 |
|------|------|
| 点击图片 | 打开03_UPLOAD文件夹 |
| 🗑 同步删除 | 删缩略图→原图送回收站 |
| ♻ 恢复 | 从回收站还原 |
| 🔄 刷新 | 按.dxlist重建缩略图 |
| ⮂ 反相 | 反相+PS黑T贴图+BW合成 |
| × | 单张缩略图送回收站 |
| 日期下拉 | 切换日期 |

## 配置
编辑 `config.py` 的 `BACK_NEW` / `FRONT_NEW`。
