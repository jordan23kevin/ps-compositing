# WB 贴图 PS 版 — 运行手册

## 前置准备
1. 安装 Python 3.x
2. 安装依赖:
   ```bash
   pip install pywin32 pillow numpy
   ```
3. 确保 Photoshop 已安装并能正常启动
4. 确保 Photoshop 中有 `正反图` 动作组，包含 `白` 和 `黑` 两个动作（仅BW合成需要）
5. 检查 `config.py` 和 `ps_batch.py` 中的路径是否正确:
   - `config.py`: `BASE_TORSO` 和 `SOURCE_BASE`
   - `ps_batch.py`: `BASE` 和 `PS_EXE`


## 完整使用流程

### 第一阶段：贴图（所有类型）
运行主脚本，处理 B、W 和 BW 类型的设计图：
```bash
python wb_sticker_ps.py
```

这一步会生成：
- **B类型**: `DXxxxx_B_白T.jpg`，`DXxxxx_B_黑T.jpg`（背面贴图，新方案）
- **W类型**: 待正图新方案完成后补充
- **BW类型**: 生成背面文件，正面待补充（为第二阶段合成做准备）


### 第二阶段：BW 合成（仅 BW 类型需要）
如果有 BW 类型的设计图，运行原来的合成脚本：
```bash
python ps_batch.py
```

这一步会：
- 读取 `DXxxxx_W_*T.jpg` 和 `DXxxxx_B_*T.jpg`
- 调用 Photoshop 的 `正反图` 动作组
- 生成最终的 `DXxxxx_白BW.jpg` 和 `DXxxxx_黑BW.jpg`


## 单个 DX 测试
可以用 `test_back_new.py` 测试单个 DX 的背图：
```bash
python test_back_new.py
```


## 背图新方案原理
1. **Python 计算位置**: 
   - 最高有效像素 y = `np.where(alpha >= 20)[0].min()`
   - 中心点 x = 所有有效像素 x 的平均值
2. **JSX 执行 PS**:
   - 用 `ArtLayer.duplicate()` 复制图层（保留透明度）
   - 缩放 30%，旋转 1°
   - 对齐：最高有效像素 → Y=570，中心点 → X=680


## 注意事项
1. **运行前请关闭 Photoshop 中已打开的文档**，避免 COM 报错。
2. 生成的图片会保存在 `DX*/03_UPLOAD/` 文件夹下。
3. 如需调整位置/大小/旋转角度，修改 `config.py` 中的 `BACK_NEW` 配置即可。
4. JSX 里必须设置 `rulerUnits = PIXELS` 并用 `.value` 取 bounds 数值。
