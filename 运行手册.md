# WB 贴图 PS 版 — 运行手册

## 前置准备
1. 安装 Python 3.x
2. 安装依赖:
   ```bash
   pip install pywin32 pillow numpy
   ```
3. 确保 Photoshop 已安装并能正常启动
4. 确保 Photoshop 中有 `正反图` 动作组，包含 `白` 和 `黑` 两个动作（仅BW合成需要）
5. 检查路径配置:
   - `config.py`: `BASE_TORSO`（胚衣路径）和 `SOURCE_BASE`（DX项目根目录）
   - `ps_batch.py`: `BASE` 和 `PS_EXE`


## 完整使用流程

### 第一阶段：贴图（所有类型）
```bash
# 处理所有DX文件夹
python wb_sticker_ps.py

# 从指定DX开始（如DX0124）
python wb_sticker_ps.py DX0124
```

生成：
- **B类型**: `DXxxxx_B_白T.jpg`，`DXxxxx_B_黑T.jpg`（背面贴图）
- **W类型**: `DXxxxx_W_白T.jpg`，`DXxxxx_W_黑T.jpg`（正面贴图）
- **BW类型**: 同时生成正面和背面共4个文件（供第二阶段合成）

运行结束会显示每张、每款、总耗时。


### 第二阶段：BW 合成（仅 BW 类型需要）
```bash
python ps_batch.py
```

读取 `_W_*T.jpg` 和 `_B_*T.jpg`，调用PS的 `正反图` 动作组，
生成 `DXxxxx_白BW.jpg` 和 `DXxxxx_黑BW.jpg`。


## 单个 DX 测试
```bash
python test_front_new.py
```
可修改脚本里的 `dx_name` 测试其他DX或背图。


## 贴图原理
1. **Python 预处理**: trim透明边距 + 缩放，保存为临时PNG
2. **位置计算**:
   - 最高有效像素y：找到一行有效像素≥50个的那行
   - 中心点x：透明度加权质心
3. **PS操作**: duplicate + 移动 + 旋转


## 关键参数说明
- **背图**: 缩放30%，旋转1°，最高有效像素→Y=570，中心点→X=680
- **正图**: 缩放13.33%（等效手动置入后设10%），旋转1°，最高有效像素→Y=612，中心点→X=888
- 缩放基准是**原始尺寸2048**


## 注意事项
1. **运行前请关闭 Photoshop 中已打开的文档**
2. 只处理 `_cut.png` 文件，其他文件（如 `DXxxxx_B.png`）会跳过
3. 如需调整位置/大小/旋转角度，修改 `config.py` 中的 `BACK_NEW` 和 `FRONT_NEW`
4. PS的 `resize(TOPLEFT)` 锚点有bug（以有效像素区域左上角为锚点，不缩放透明区域），
   所以用Python预处理避开
