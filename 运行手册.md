# WB 贴图 PS 版 — 运行手册

## 前置准备
1. 安装 Python 3.x
2. 安装依赖:
   ```bash
   pip install pywin32 pillow numpy
   ```
3. 确保 Photoshop 已安装并能正常启动
4. 确保 Photoshop 中有 `正反图` 动作组，包含 `白` 和 `黑` 两个动作（仅BW合成需要）
5. 检查路径配置:
   - `config.py`: `BASE_TORSO`（胚衣路径）和 `SOURCE_BASE`（DX项目根目录）
   - `ps_batch.py`: `BASE` 和 `PS_EXE`


## 完整使用流程

### 第一阶段：贴图（所有类型）
运行主脚本，处理 B、W 和 BW 类型的设计图：
```bash
python wb_sticker_ps.py
```

这一步会生成：
- **B类型**: `DXxxxx_B_白T.jpg`，`DXxxxx_B_黑T.jpg`（背面贴图）
- **W类型**: `DXxxxx_W_白T.jpg`，`DXxxxx_W_黑T.jpg`（正面贴图）
- **BW类型**: 同时生成正面和背面共4个文件（供第二阶段合成）


### 第二阶段：BW 合成（仅 BW 类型需要）
如果有 BW 类型的设计图，运行原来的合成脚本：
```bash
python ps_batch.py
```

这一步会：
- 读取 `DXxxxx_W_*T.jpg` 和 `DXxxxx_B_*T.jpg`
- 调用 Photoshop 的 `正反图` 动作组
- 生成最终的 `DXxxxx_白BW.jpg` 和 `DXxxxx_黑BW.jpg`


## 单个 DX 测试
用 `test_front_new.py` 测试单个 DX（默认测试DX0086正图）：
```bash
python test_front_new.py
```
可修改脚本里的 `dx_name` 测试其他DX或背图。


## 贴图原理
1. **Python 计算位置**: 
   - 最高有效像素 y：找到一行有效像素≥50个的那行
   - 中心点 x：透明度加权质心 `(x * alpha).sum() / alpha.sum()`
2. **JSX 执行 PS**:
   - 用 `ArtLayer.duplicate()` 复制图层（保留透明度）
   - 缩放、旋转
   - 对齐：最高有效像素 → Y，中心点 → X


## 关键参数说明
- **背图**: 缩放30%，旋转1°，最高有效像素→Y=570，中心点→X=680
- **正图**: 缩放13.33%（等效手动置入后设10%），旋转1°，最高有效像素→Y=612，中心点→X=888
- 缩放基准都是**原始尺寸2048**（用duplicate方式，不经过PS置入）


## 注意事项
1. **运行前请关闭 Photoshop 中已打开的文档**，避免 COM 报错 "application busy"。
2. 生成的图片会保存在 `DX*/03_UPLOAD/` 文件夹下。
3. 如需调整位置/大小/旋转角度，修改 `config.py` 中的 `BACK_NEW` 和 `FRONT_NEW` 配置即可。
4. JSX 里必须设置 `rulerUnits = PIXELS` 并用 `.value` 取 bounds 数值。
5. 不要用 Python 差异检测来判断贴图位置（旋转后边角半透明像素会干扰），应直接在JSX里读取图层bounds验证。


## 验证结果
- DX0086 背图（白T/黑T）：效果良好，位置精确
- DX0086 正图（白T/黑T）：贴图大小153×234 ≈ 手动155×234，位置精确
- DX0086 BW合成图：效果良好
