# WB 贴图 Photoshop 版 — 项目架构

## 概述
将原本美图秀秀的 WB 贴图流程迁移到 Photoshop，使用 JSX + Python win32com 自动化实现。

## 文件结构

```
ps/
├── config.py              # 配置文件（路径、坐标、缩放、旋转）
├── wb_sticker_ps.py       # 主程序入口（第一阶段：贴图）
├── ps_batch.py            # 原来的合成脚本（第二阶段：BW合成）
├── test_front_new.py      # 单个DX测试脚本
├── jsx/
│   └── place_design.jsx   # 通用贴图JSX（正图/背图共用）
├── 运行手册.md            # 使用说明
├── 问题记录.md            # 开发过程中遇到的问题及解决方案
└── 项目架构.md            # 本文档
```

## 核心算法（复刻美图秀秀 calc_offset_back）

### 贴图位置计算（calculate_sticker_position）

1. **最高有效像素 y**：alpha >= 20 的像素中，从上往下找到**一行有效像素≥50个**的那行
   （防止顶部噪点干扰）
2. **中心点 x**：透明度加权质心 `(x * alpha).sum() / alpha.sum()`
   （不是简单平均值，用alpha做权重）

### PS操作流程（place_design.jsx）

1. 用 `ArtLayer.duplicate()` 复制图层（保留原始尺寸和透明度）
2. `resize(scalePercent)` 缩放（百分比，相对于原始尺寸2048）
3. `rotate(rotation)` 旋转
4. 设置 `rulerUnits = PIXELS`，用 `.value` 取 bounds 数值
5. 移动图层：最高有效像素 → Y，中心点 → X

## 关键参数（config.py）

### 背图 (BACK_NEW)
```python
BACK_NEW = {
    "torso_white": "白背2.jpg",
    "torso_black": "黑背2.jpg",
    "scale_percent": 30,      # 缩放30%（相对原始2048）
    "rotation": 1,            # 向右旋转1°
    "target_center_x": 680,   # 贴图中心点目标X
    "target_top_y": 570,      # 贴图最高有效像素目标Y
}
```

### 正图 (FRONT_NEW)
```python
FRONT_NEW = {
    "torso_white": "白正2.jpg",
    "torso_black": "黑正2.jpg",
    "scale_percent": 13.33,    # 实测等效于手动置入后设10%
    "rotation": 1,            # 向右旋转1°
    "target_center_x": 888,   # 贴图中心点目标X
    "target_top_y": 612,      # 贴图最高有效像素目标Y
}
```

> **关于正图13.33%的说明**：
> 用户手动操作是"置入贴图后设10%"。但PS置入时会自动缩放，
> 实测手动"10%"等效于相对原始尺寸(2048)的13.33%。
> 用 duplicate 方式（不经过PS置入）需直接用13.33%才能复刻手动效果。
> 验证：生成贴图有效区域 153×234px，手动测量 155×234px，吻合。

## 两阶段处理流程

### 第一阶段：贴图（wb_sticker_ps.py）

1. 扫描 `SOURCE_BASE` 下所有 `DX*` 文件夹
2. 从 `DX*/02_REM_BG/` 读取 PNG
3. 根据文件名分类：BW / W / B
4. 生成贴图：
   - **B类型**: 背图新方案，生成 `DXxxxx_B_白T.jpg`，`DXxxxx_B_黑T.jpg`
   - **W类型**: 正图新方案，生成 `DXxxxx_W_白T.jpg`，`DXxxxx_W_黑T.jpg`
   - **BW类型**: 同时生成正面和背面共4个文件（供第二阶段合成）

### 第二阶段：BW合成（ps_batch.py，仅BW类型需要）

1. 从 `DX*/03_UPLOAD/` 读取 `_W_*T.jpg` 和 `_B_*T.jpg`
2. 调用 Photoshop 的 `正反图` 动作组
3. 生成 `DXxxxx_白BW.jpg` 和 `DXxxxx_黑BW.jpg`
