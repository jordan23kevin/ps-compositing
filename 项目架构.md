## 两阶段处理流程

### 第一阶段：贴图（wb_sticker_ps.py）
```bash
python wb_sticker_ps.py [起始DX]
```
- 从 `02_REM_BG/` 读取 `_cut.png`（跳过含 `_黑` 的黑版文件）
- Python trim透明边距 + 缩放 → 临时PNG
- PS duplicate + move + rotate → `03_UPLOAD/`

### 第二阶段：BW合成（ps_batch.py）
```bash
python ps_batch.py [起始DX]
```
- 用 PS 动作 `正反图` 合成正背图

## 反相 → 黑T流程

| 步骤 | 触发 | 执行 |
|------|------|------|
| 1. 反相原图 | ⮂ 按钮 | PIL反相RGB，另存为 `DXxxxx_黑X_cut.png` |
| 2. PS黑T贴图 | 后台启动 | `process_black.py`：黑版→黑色胚衣→B_黑T/W_黑T |
| 3. BW合成 | 同上 | PS动作 `正反图 > 黑` → 黑BW |
