## 1. 项目目标

对 WB贴图 输出的 T恤正反面图片（`白正2_副本.jpg`、`白背2_副本.jpg` 等），通过 Photoshop 的 `正反图` 动作合成正反一体图，输出 `白BW.jpg` / `黑BW.jpg`。

### 上下游关系

```
WB贴图 → DXxxxx\白正2_副本.jpg  →  PS合成 → DXxxxx\白BW.jpg
                \白背2_副本.jpg                        \黑BW.jpg
                \黑正2_副本.jpg
                \黑背2_副本.jpg
```

## 3. 核心脚本

### `ps_batch.py` — 主批处理 v1.0

**流程：**
1. 扫描 `D:\Semems\1AI` 下所有 DX 文件夹
2. 每文件夹先处理白T → 再处理黑T
3. 跳过已存在 `白BW.jpg` / `黑BW.jpg` 的文件夹

**单次处理：**
- `ps_open_both()` → 一次命令行打开正背两张图
- `wait_ps_docs(2)` → 轮询等待 2 个文档打开（0.5s 间隔，不用固定 sleep）
- 激活 背 文档
- `DoAction('白'/'黑', '正反图')` → 运行动作
- `Export(ExportAs=2, Quality=100)` → 保存 JPEG（不弹窗）
- `Close(2)` → 关闭所有文档

**速度优化：**
- 两张图同时打开（一次命令行传两个文件）
- 用 COM 轮询等待代替固定 sleep

## 6. 保存方式对比

| 方式 | 结果 |
|------|------|
| `SaveAs` → JPEG 选项弹窗 ❌ | 会弹窗，需要手动点确定 |
| `Export(ExportOptionsSaveForWeb)` ✅ | **不弹窗，直接保存** |
| `DoJavaScript` → `doc.saveAs()` | 有时活动文档不可用 |

最终方案：**Export + ExportOptionsSaveForWeb**（Quality=100）
